Controls:

RECTANGLE MANIPULATION-------------------
Right-arrow: increase width of rectangle
Left-arrow: decrease width of rectangle
Up-arrow: increase height of rectangle
Down-arrow: decrease height of rectangle
R-key: rotate the rectangle by PI/8 radians (22.5 degrees)

SCENE MANIPULATION----------------------
S-key: switch to cursor to select already placed objects
P-key: switch to placing player (must contain player and target to use level in game)
T-key: switch to placing target (must contain player and target to use level in game)
K-key: switch to killzone type
D-key: delete current object, unless it is only object in scene
